クラコの技術を生かした製品は、ホテルやレストランの業務用厨房に「安全」「清浄」「快適」を維持するためには欠かせません。

各製品名を押すとページ内で移動します。

従来の１/４（当社比）の静圧が、消費電力の削減（５～30％削減）、騒音の低減（当社比50%低減）を可能にしました。

  1. 驚異的な低静圧化（20Pa）により、ファンモーターの小型化、ランニングコストの削減、低騒音化などを容易にします。
  2. エアロフォイル理論に基づいた独自のフィルタ羽根形状が、静圧を極めて低くしながらグリス除去率80％（静圧20Pa時）以上、グリス付着率4%以下（自社測定値）を実現しました。
  3. フィルタとチャンバーはすべてステンレス製のため、錆に強く、長時間の使用に耐えます。
  4. 東京消防庁はじめ、国内各地の火災予防条例およびグリス除去装置に関する技術基準に適合しており、(一社)日本厨房工業会の認定品です。

業界No.１の販売実績が厨房の安全と快適を応援します。

  1. 特殊な構造のチェックプレートの組合せがグリス除去率を約80％（静圧69Pa時）までアップさせました。
  2. 負圧室を設けることにより、長時間連続使用の場合でも目詰まりしにくく、安定した高グリス除去率と低グリス付着率を維持します。
  3. フィルタとチャンバーはすべてステンレス製のため、錆に強く、長時間の使用に耐えます。
  4. フィルタはワンタッチで分解可能なため、手軽に洗浄できます。
  5. 東京消防庁はじめ、国内各地の火災予防条例およびグリス除去装置に関する技術基準に適合しており、(一社)日本厨房工業会の認定品です。

東京消防庁　技術基準適合品、国土交通省「ダンパー構造基準」適合品がいっそうの安心をお届けします。

  1. 羽根形状の改良により従来型の1/2（当社比）の静圧を実現し、より経済的になりました。
  2. 熱感知部に当社独自の高精度温度ヒューズを使用し、作動精度が飛躍的に向上しました。
  3. 1.5mmステンレス鋼板を使用。錆びに強く、長時間の使用に耐えます。
  4. ダンパー開閉と風量調整が容易にできます。
  5. 東京消防庁「業務用厨房設備に付属する天蓋及び排気ダクト」技術基準適合品です。
  6. 国土交通省「ダンパー構造基準」適合品です。(建設省告示2563号、2565号の「温度ヒューズ連動閉鎖装置の作動および不作動」ならびに「防火ダンパー漏煙」試験に合格[(財)日本建築総合試験所])

電子の見張り番が、24時間自動で厨房火災をしっかりガードします。

  1. 消火剤容器格納箱が、従来型の約1／2(当社比)の容積になり、厨房スペースが有効に活用できるようになりました。
  2. ダクト内の温度状況を「正常」「警報」「消火」の３段階で監視。操作盤の表示灯やブザーで危険状況を予報する安全設計になっています。
  3. 充実したセルフチェック機能が、常に「センサー」「起動装置」や電気系統を監視。高い安全性を確保しています。
  4. シティーエース25”は(財)日本消防設備安全センターの技術基準に合格した性能評定合格品です。

クラコのエアロ・ギミックは独自の構造を持つ「ギミックコア」の働きで油脂分を高効率で除去し、安全性・経済性・利便性などのさまざまなメリットを提供します。

  1. グリス除去率が非常に高いため、ダクト清掃は従来型フードの場合に対し半分以下の回数ですみます。
  2. 洗浄用ボタンを押せばノズルから湯（または水）が噴出して機内を自動洗浄します。
  3. グリス除去の心臓部である「ギミックコア」は簡単に取り外し洗浄ができます。取り外し洗浄回数は一般フィルタの1/10以下の頻度でＯＫ。
  4. エアロ・ギミックに火が入った場合、自動的にダンパーが閉鎖し、湯（または水）が噴出してダクト内への火炎伝送を防止する設定も可能です（オプション）。
  5. エアロ・ギミックは(一社)日本厨房工業会　グリスエクストラクター認定品です。
